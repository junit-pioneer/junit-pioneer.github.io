    Introduction

    Test shall be fast. To get a feeling how fast or slow a specific test is, the @Stopwatch annotation can be used. It will measure elapsed time and pass the result to the TestReporter, which will publish it.

    Usage

    To measure elapsed time just add the @Stopwatch annotation to a class or, like in the following example, to a method:

    @Stopwatch
@Test
void test() {
    // Some test
}
